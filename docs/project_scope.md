# Project Scope

## Problem

Road traffic incidents have a spatial component that can be analyzed using geospatial databases, spatial SQL and GIS tools.

## Study Area

Popayán, Cauca, Colombia.

## Main Question

Where are road traffic incidents concentrated in Popayán, and which areas show higher spatial criticality?

## Expected Outputs

- Clean geospatial dataset of road traffic incidents.
- PostgreSQL/PostGIS spatial database.
- Spatial SQL queries.
- QGIS maps.
- Technical documentation.
- GitHub repository for portfolio use.

## Initial Technical Focus

- Data cleaning with Python.
- Spatial data modeling with PostGIS.
- Spatial analysis with SQL.
- Cartographic visualization with QGIS.
